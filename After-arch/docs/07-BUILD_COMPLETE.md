## 🚀 Getting Started

### Quickest Start (30 seconds)

1. **Set up environment:**
   ```bash
   cd backend && cp .env.example .env  # Add Supabase credentials
   cd ../frontend && cp .env.example .env
   cd ../ml-service && cp .env.example .env
   ```

2. **Start services (Windows):**
   ```bash
   start-backend.bat      # Terminal 1
   start-frontend.bat     # Terminal 2
   start-ml-service.bat   # Terminal 3
   ```

3. **Access application:**
   - Frontend: http://localhost:3000
   - Backend: http://localhost:5000
   - ML: http://localhost:5001

4. **Create account and login:**
   - Register with email/password
   - Start marking attendance

**Total time to running system: 5 minutes!**
